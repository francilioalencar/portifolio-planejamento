# portifolio-planejamento
Repositório portifolio linguagem JAVA, aqui foi . 

<p>Versão disponibilizada e desenvolvida em Java com  framework Spring Boot.</p>
<ul>

  <li>Banco de dados Mysql;</li>

</ul>
